# Cenu izmaiņu izsekošanas sistēma

## Projekta apraksts

Cenu izmaiņu izsekošanas sistēma ir tīmekļa lietotne, kas ļauj lietotājiem pievienot interesējošās preces un automātiski sekot to cenu izmaiņām dažādos e-veikalos. Sistēma palīdz centralizēti pārvaldīt cenu izmaiņas vairākos veikalos vienlaikus, ietaupot laiku un palīdzot pieņemt izdevīgākus pirkuma lēmumus.

## Pamatproblēma

Mūsdienās lielākā daļa iepirkšanās notiek tiešsaistē, un vienas un tās pašas preces cena dažādos e-veikalos var būtiski atšķirties. Produktu cenas mainās ļoti bieži, un manuāla cenu pārbaude prasa daudz laika un uzmanības. Nav daudz risinājumu, kas ļautu viegli sekot precēm vairākās vietnēs vienlaikus un analizēt cenu vēsturi.

## Sistēmas mērķis

Nodrošināt ērtu, drošu un automatizētu cenu novērošanas procesu, kas lietotājiem sniedz pārskatāmu informāciju par cenu dinamiku un paziņojumus par izmaiņām. Sistēma apvieno cenu pārbaudes, paziņojumu un datu glabāšanas funkcijas vienā pieejamā risinājumā.

## Galvenās nakotnes funkcionalitātes

- Lietotāju reģistrācija un autentifikācija
- Preču pievienošana uzraudzības sarakstam
- Automātiska cenu pārbaude vairākos e-veikalos
- Paziņojumi par cenu izmaiņām
- Cenu izmaiņu vēstures pārskats
- Cenu dinamikas vizualizācija un analytics
